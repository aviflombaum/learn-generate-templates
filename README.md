# `learn generate` Templates

This repo will contain the canonical list of Learn.co template types
that will be available to the `learn generate` command.

The current list of templates is:

1. fundamental-ruby
2. command-line
3. sql
4. activerecord
5. rake
6. rack
7. sinatra-classic
8. sinatra-mvc
9. js
10. front-end
11. kids
